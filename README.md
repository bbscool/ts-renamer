# ts-renamer

## Install

```bash
git clone https://github.com/rndomhack/ts-renamer.git
cd ts-renamer
npm i
cp ./sample/settings.sample.json ./sample/settings.json
```

## Usage

Linux or OSX
```bash
./scripts/main.sh foo.ts
./scripts/main.sh /foo/bar/baz
```

Windows
```cmd
.\scripts\main.bat foo.ts
.\scripts\main.bat C:\foo\bar\baz
```
